# Shilab Inventory Monitor Bot

בוט פייתון חכם שמבצע ניטור (Monitoring) בזמן אמת אחר זמינות מוצרים באתר "שילב". 

## 🚀 תכונות עיקריות
- **סריקה אוטומטית:** הבוט בודק את זמינות המוצרים במרווחי זמן קבועים.
- **התראות מיידיות:** שליחת הודעה ברגע שהמלאי משתנה מ"אזל מהמלאי" ל"זמין לרכישה".
- **תמיכה מרובה משתמשים:** ניהול רשימת תפוצה של משתמשים לקבלת התראות בטלגרם.
- **יציבות:** תוכנן לרוץ 24/7 על שרת ענן (Google Cloud) באמצעות `screen`.

## 🛠 טכנולוגיות בשימוש
- **Python:** שפת הפיתוח הראשית.
- **Telebot (pyTelegramBotAPI):** לתקשורת מול משתמשי טלגרם.
- **Requests & BeautifulSoup:** לצורך משיכת נתונים וניתוח ה-HTML של אתר שילב.

## 📋 מבנה הפרויקט
- `shilab_monitor.py`: הליבה של הבוט, אחראית על הסריקה והבדיקה מול האתר.
- `shilab_multi_bot.py`: ניהול הבוט בטלגרם ושליחת הודעות למשתמשים.
- `users.txt`: רשימת המשתמשים הרשומים לקבלת התראות.
- `.env`: קובץ הגדרות מאובטח השומר את הטוקנים של הבוט.

## ⚙️ התקנה והרצה
1. שכפל את המאגר:
   `git clone [URL_OF_YOUR_REPO]`
2. התקן את הספריות הנדרשות:
   `pip install -r requirements.txt`
3. הגדר את הטוקנים שלך בקובץ `.env`.
4. הרץ את הבוט בסביבת `screen` כדי להבטיח פעולה רציפה:
   `screen -S shilab_bot python shilab_monitor.py`
